# Weather app

## How to run

### Sails API
`cd weather-api`

`npm i`

Node 23.8.0 was used for development

`sails lift` to start dev server

`.env.example` shows requirements (mongo URI)

### Nuxt frontend

`cd weather-api`

`npm i`

`npm run dev` or `npm run build`

`.env.example` shows requirements (API url)
